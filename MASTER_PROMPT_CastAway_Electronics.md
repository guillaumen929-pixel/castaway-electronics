## TECHNICAL REQUIREMENTS

- **Framework:** Plain HTML, CSS, and vanilla JavaScript — OR React with Vite — your choice, but keep it clean and production-ready
- **Responsive:** Fully mobile-first responsive design (320px → 1920px)
- **Performance:** Optimize for fast load — no heavy external dependencies
- **Accessibility:** Semantic HTML, proper ARIA labels, keyboard navigation
- **Icons:** Use Lucide icons, HeroIcons, or inline SVG — no paid icon libraries
- **Images:** Use placeholder images from `https://placehold.co` or `https://picsum.photos` with descriptive alt text. Label where real photos should be swapped in. Suggest themes: recycling facility, circuit boards, technician with electronics, South Florida skyline
- **Animations:** Subtle scroll-reveal animations on section entry (CSS only or lightweight JS)
- **No external CSS frameworks required** — but Tailwind CSS is acceptable if preferred
- **File Structure:**
```
/
├── index.html
├── about.html
├── services.html
├── contact.html
├── css/
│   └── styles.css
├── js/

    └── (placeholder images)
```
